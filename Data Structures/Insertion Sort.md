# Insertion Sort

## Example Array 1
Array -> [22,27,16,2,18,6]

### Stages

1. Array -> [22,27,16,2,18,6]
2. Array -> [16,22,27,2,18,6]
3. Array -> [2,16,22,27,18,6]
4. Array -> [2,16,18,22,27,6]
5. Array -> [2,6,16,18,22,27]

### Big-O Notation

Big o notation is *n^2*

### Time Complexity

#### Average Case
n^2

#### Best Case
n

#### Worst Case
n^2

### Time Complexity For 18 After Sorting
Average Case
